# Regras Globais

Regras absolutas que se aplicam a TODOS os criativos, independente de nicho ou produto.
Cada regra foi registrada a partir de feedback do usuário.
Nenhuma regra pode ser ignorada ou sobrescrita por instrução de sessão.

---

[2026-04-07] [GLOBAL] Nunca usar travessão (—) em copy de criativo. Substituir por vírgula, ponto ou reescrita.

[2026-04-07] [GLOBAL] O preço sempre visível e destacado. Nunca esconder o investimento no criativo.

[2026-04-07] [GLOBAL] Toda copy deve estar em português brasileiro perfeito. Zero anglicismos sem necessidade.

[2026-04-07] [GLOBAL] Máximo de 1 emoji por criativo. Preferivelmente nenhum. Nunca usar emoji decorativo.

[2026-04-07] [GLOBAL] Nunca usar fundo branco puro (#FFFFFF sem textura ou gradiente). Todo fundo deve ter personalidade visual: textura, gradiente, cor com profundidade.
